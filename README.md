# iplan（交互制定即时计划APP）
iplan是一款可以写计划的app，但又不只是写计划的app,他的出世，不是为了让人们制定几天，甚至长久计划，而是制定即时计划。什么？懒得写计划？一个闹钟就够了？不不不，我们不是让你自己制定计划。而是：女朋友中午给男朋友制定3点起床并写作业计划；男朋友中午给女朋友制定2点喝水，5点喝水计划；妈妈上午给小明制定中午1点吃药计划。这是一款可以用来改变你爱的人的app,也可以被改变。
